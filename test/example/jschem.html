Experimental WISE schematic entry.

Here is the homepage of the project!

This applets are for test and training purposes.


Press the corresponding button to start an experiment.
A low pass filter example:
The circuit contains of a capacitor (C1) parallel to the output port and a resistor (R1) between input and output. The cutoff frequency is defined by the value of R1 and C1. On the input an AC source with 1V is attached.
(demo with AC, 1e8 stop, 1 start, 20 points)

A high pass filter example:
The circuit contains of a resistor (R1) parallel to the output port and a capacitor (C1) between input and output. The cutoff frequency is again defined by the value of R1 and C1. On the input an AC source with 1V is attached.
(demo with AC, 1e8 stop, 1 start, 20 points)

A (too) simple amplifier example:
The circuit contains a transistor stimulated by a sine source.
(demo with TRAN 1e-5 step, 1e-3 stop)

A simple power supply example:
The circuit contains a diode for an simple AC/DC converter.
(demo with TRAN, 5e-2 stop, 1e-7 step)

A MOS diode:
The circuit contains a N-channel MOSFET acting as a 'MOS diode'.
(demo with DC, 0 start, 2 stop, 0.02 step)

A MOS current mirror:
The circuit contains two N-channel MOSFET acting as a current mirror.
(demo with DC, 0 start, 1e-4 stop, 1e-5 step)

An ideal, noninverting amplifier:
The circuit contains a voltage dependent voltage source configured as amplifier.
(demo with TRAN 1e-5 step, 1e-3 stop)
